# Raw Sources

本目录用于存放用户提供的原始资料，例如文章摘录、需求原文、会议记录、截图说明或外部文档导出文件。

## 规则

- 原始资料是事实来源；除修正导入错误外，不改写已归档的来源内容。
- 新资料应使用可识别的文件名；必要时可按主题建立子目录。
- 对外部 URL，如果无需保存全文，可创建一个来源记录文件，写明链接、获取日期和用途。
- 归档新资料后，由代理读取来源并将整理结果写入 `../wiki/`，同时更新 `../wiki/index.md` 与 `../wiki/log.md`。
